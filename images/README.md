# 画像アセット

サイトで使用するイラスト系SVGを格納するディレクトリ。テクノ・ミニマルのサイトデザインに合わせて**手書きSVG**で揃えています。

> ブランドアセット（ロゴ・ファビコン・ロックアップ・OGP・brand-tokens.css）は [../assets/](../assets/) に格納しています（ブランドガイドライン v1, 2026-05-12 以降）。本ディレクトリにはサービスアイコンとヒーロービジュアルのみ残しています。

## 配置済みアセット

### サービスアイコン（新ロゴ意匠に統一、2026-05-12）

| ファイル | 紐づくサービス | 構造 |
|---|---|---|
| `icons/consult.svg` | SVC.001 IT相談 | 4隅ブラケット + 吹き出し+三点 + シアン対角の発話しっぽ |
| `icons/tenko.svg` | SVC.002 自動点呼の運用支援 | 4隅ブラケット + 円 + チェック（長辺がシアン対角充填） |
| `icons/security.svg` | SVC.003 セキュリティチェック対応 | 4隅ブラケット + 盾形 + シアン対角の検証マーク |
| `icons/dev.svg` | SVC.004 個別開発・運用支援 | 4隅ブラケット + `< >` + シアン対角のスラッシュ |

統一ルール:
- `viewBox="0 0 200 200"` `shape-rendering="geometricPrecision"`
- 主線 `stroke="currentColor"` `stroke-width="6"` `fill="none"`（CSS 側 `color: var(--cyan)` 等で発色を制御）
- 各アイコンに必ず**シアン対角の充填要素**を 1 つ含める（新ロゴの DNA を継承、`fill="#00d4ff"`）
- 数値はすべて 4 の倍数で構築

サービスアイコンも本体には [../index.html](../index.html) にインラインSVGとして埋め込んでいます。

## ラスター画像（写真・イラスト）の追加について

現状のテクノ・ミニマル設計は**写真を入れない方が一貫性が保てる**ため、写真追加は慎重に検討してください。

追加する場合:
1. AI画像生成ツール（ChatGPT、Midjourney、Adobe Firefly等）でPNG/JPGを生成
2. このディレクトリに配置（例: `hero-bg.png`, `staff.jpg`）
3. `index.html` および `style.css` に `<img>` タグまたは `background-image` で組み込み
4. 商用利用可能なライセンスを必ず確認
